# Введение
Это телеграм-бот-парсер сайта колледжа. Сейчас бот присылает изменения в расписании для выбранной группы.

# Live-парсинг расписания занятий (mpt.ru/raspisanie)
На странице `https://mpt.ru/raspisanie/` расписание хранится в HTML этой же страницы, а переходы по отделениям/группам работают через якоря вида `#hash`.

Из-за этого **отдельного HTTP API для конкретной группы нет**: HTTP-запрос всегда к одной странице, а нужный блок выбирается по `id` в HTML.

## Что использовать
Добавлен скрипт `src/mpt_schedule_client.py`, который:
- загружает страницу расписания;
- собирает якоря отделений/групп;
- находит нужные блоки (например, `09.02.06 СА` и `СА-1-23; СА-11/1-24`);
- печатает deep-link и текст расписания из соответствующего блока.

## Пример запуска
```bash
python src/mpt_schedule_client.py \
  --department "09.02.06 СА" \
  --group "СА-1-23; СА-11/1-24"
```
